## Syntax

```
P12 TO PEM (pfx;password;certificate;privateKey)
```

Parameter|Type|Description
------------|------------|----
pfx|BLOB|P12 certificate (in)
password|TEXT|Password (in)
certificate|TEXT|Certificate (out)
privateKey|TEXT|Private key (out)
